# Arrow Quest — ローグライトアーチェリーゲーム

ブラウザで動く3Dローグライト風アーチェリーゲームです。  
「Arrow a Row」にインスパイアされたオリジナル作品として制作しました。

---

## 🎮 起動方法

`index.html` をブラウザ（Chrome / Edge 推奨）で開いてください。

> ⚠️ ローカルファイルから ES Module を `import` するため、  
> **VSCode の Live Server 拡張** や **http-server** などを使って  
> `http://localhost:xxxx` 経由で開くことをお勧めします。  
> ダブルクリックでの直接起動は `file://` プロトコルとなり、CORS エラーが発生する可能性があります。

---

## 🕹️ 操作方法

| 操作 | 内容 |
|------|------|
| `W/A/S/D` | 移動 |
| マウス移動 | 照準 |
| 左クリック | 矢を発射 (連射) |
| `Space` | 回避ダッシュ |
| `Esc` | ポーズ / 再開 |

---

## ⚔️ ゲームの進め方

1. タイトル画面で「ゲームスタート」を押す
2. 敵を全滅させるとラウンドクリア
3. 3択のアップグレードから1つを選んで強化
4. Round 7 で大型ボスが登場
5. ボスを倒すとゲームクリア！

---

## 🏹 敵の種類

| 種類 | 説明 |
|------|------|
| 近接雑魚 (赤) | まっすぐ追いかけてくる |
| 遠距離雑魚 (紫) | 距離を保ちながら魔法弾を放つ |
| 突進雑魚 (橙) | 予備動作後に猛スピードで突進 |
| ボス (灰) | HPが減るごとにフェーズが変わる大型敵 |

## 📁 ファイル構成

```
弓矢のゲーム２/
├── index.html    … エントリーポイント・UIテンプレート
├── style.css     … 全UIスタイル
├── main.js       … ゲームループ・全体管理
├── world.js      … 3Dマップ生成 (地面・木・岩・遺跡)
├── player.js     … プレイヤー操作・射撃・ダッシュ
├── enemies.js    … 敵AI (近接・遠距離・突進・ボス)
├── combat.js     … 矢・弾・当たり判定・エフェクト
├── upgrades.js   … アップグレード定義・選択ロジック
├── ui.js         … UIマネージャー (スクリーン管理)
└── README.md     … このファイル
```

## 🛠️ カスタマイズのヒント

- **敵の強さを調整**: `enemies.js` の各クラスコンストラクタ内の数値を変更
- **新アップグレード追加**: `upgrades.js` の `ALL_UPGRADES` 配列に追記
- **ラウンド数変更**: `main.js` の `TOTAL_ROUNDS` 定数を変更
- **マップの広さ変更**: `world.js` の `PlaneGeometry` サイズを変更
- **カメラ角度変更**: `main.js` の `_setCameraOffset()` の Y/Z 値を変更
